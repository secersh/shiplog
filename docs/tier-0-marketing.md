# Tier 0 Marketing: Ghost Mode

Start date: 2026-06-14
End date: 2026-06-20

Tier 1 starts: 2026-06-21

## Purpose

Tier 0 tests what happens when ShipLog is public, usable, and discoverable, but not actively promoted.

This phase creates a clean baseline before intentional marketing begins. Any visitors, installs, repository activations, or revenue during this period should come from organic discovery, existing public links, search indexing, GitHub discovery, or incidental traffic.

## Rule

Organic discovery only.

Do not run active distribution during Tier 0. The product can exist publicly, but it should not be pushed into audiences yet.

## Allowed

Keep the product live at `https://shplog.dev`.

Keep the GitHub repository public.

Keep public docs, logs, legal pages, and baseline metrics up to date.

Fix bugs that block product use.

Improve onboarding, copy, reliability, and measurement.

Record metrics and observations.

Respond naturally if someone discovers the project and contacts you first.

## Not Allowed

No Product Hunt launch.

No Hacker News launch post.

No Reddit launch post.

No dev.to launch article.

No LinkedIn launch post.

No X launch thread.

No cold outreach.

No community posting.

No paid ads.

No coordinated asks for installs, feedback, upvotes, or shares.

## What To Measure

Visitors.

GitHub repository views and stars.

GitHub App installs.

Activated repositories.

Release notes generated.

Customers.

Revenue.

MRR.

Support messages or inbound feedback.

## Daily Routine

Check production health.

Check GitHub App install and repository activation flow.

Check release-note generation.

Record notable metrics or events.

Fix launch-blocking bugs only.

Avoid turning fixes into new feature work unless the product is broken or confusing.

## End Of Tier 0

At the end of 2026-06-20, write a short summary in `logs/week-2.md`:

- What changed
- What broke
- What was fixed
- What organic traffic appeared
- Whether anyone installed or used ShipLog without active marketing
- Baseline metrics before Tier 1 starts

Tier 1 marketing begins on 2026-06-21.
